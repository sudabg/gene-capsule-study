# auto-researcher 研究任务

## 任务
实现从 Agent 执行轨迹中自动抽取可复用经验模式的完整系统

## 研究内容
1. 文献调研 - ACL/EMNLP/ICLR/NeurIPS 相关论文
2. 算法设计 - 轨迹预处理、模式识别、抽象泛化
3. 代码实现 - TrajectoryParser, PatternExtractor, Generalizer, CapsuleBuilder
4. 实验验证 - 测试数据集、评估指标、实验结果
5. 文档完善 - README、API 文档、技术报告

## 输出要求
- 完整可运行的代码
- 文献综述 + 算法设计 + 实验报告
- 测试数据 + 实验结果
- 端到端演示示例

## 注意事项
